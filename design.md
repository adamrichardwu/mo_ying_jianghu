# 墨影江湖 - 设计文档

## 1. 项目概述

**项目名称：** 墨影江湖

**游戏类型：** 2D 横板 / 漫画风 / 武侠 / 回合制 / 养成收集

**核心定位：**  
一款以“修炼、收集武学、游历江湖、策略战斗”为核心的 2D 横板漫画风武侠游戏。  
玩家将扮演一名初入江湖的习武者，通过游历地图、挑战敌人、修炼内外功、收集武学秘籍，逐步构建自己的武学体系。

**设计原则：**
- 不预设复杂主线剧情，优先搭建可循环游玩的核心玩法
- 强调“武学搭配”与“成长路线”
- 战斗采用易理解但有深度的回合制系统
- 世界观借鉴传统武侠题材的共性设定，不直接使用已有作品的专有角色、门派、招式名称

## 2. 美术与表现风格

## 2.1 视觉风格
- 2D 横板表现
- 漫画风人物造型与立绘
- 场景偏写意国风，兼顾轻松与江湖感
- 战斗演出强调招式名称、动作分镜、气劲特效

## 2.2 氛围关键词
- 江湖
- 门派
- 修行
- 游历
- 秘籍
- 擂台
- 奇遇
- 恩怨

## 2.3 UI 风格
- 古风卷轴、木牌、宣纸元素
- 重要武学和装备使用图鉴式展示
- 战斗界面突出“招式、真气、状态、连携”

## 3. 核心玩法循环

玩家的基础循环如下：

1. **游荡江湖**
   - 在不同区域移动、探索、触发事件
   - 遇敌、奇遇、商人、隐士、秘籍线索

2. **战斗获胜**
   - 获取修为、铜钱、材料、武学残页、装备
   - 装备掉落受敌人掉落表、场景与稀有度共同影响

3. **修炼成长**
   - 提升角色属性
   - 练习内功、轻功、外功
   - 突破境界或阶段

4. **收集与搭配**
   - 获得新的武学
   - 调整轻功 / 内功 / 外功组合
   - 更换武器 / 衣服 / 饰品 / 护腕 / 鞋子 / 帽子 / 戒指
   - 优化战斗流派

5. **挑战更高难度内容**
   - 更强敌人
   - 更高级地图
   - 更稀有秘籍与装备

## 4. 角色成长系统

## 4.1 角色基础属性
建议采用以下基础属性：

- **气血**：生命值
- **真气**：释放武学与内功效果所需资源
- **臂力**：影响部分外功伤害与招架能力
- **身法**：影响速度、闪避、先手
- **根骨**：影响气血、防御、抗性
- **悟性**：影响修炼效率、学习条件、特殊领悟概率

## 4.2 战斗派生属性
- 外功攻击
- 内劲攻击
- 外功防御
- 内功防御
- 命中
- 闪避
- 暴击
- 招架
- 速度
- 真气恢复
- 状态抗性

## 4.3 成长方式
- 战斗获取修为
- 打坐修炼
- 使用秘籍学习新武学
- 重复使用招式提升熟练度
- 通过特殊事件获得顿悟
- 装备与武学组合形成流派成长

## 4.4 当前原型中的装备定位
为避免“武学负责一切数值，装备只做展示”的问题，首版原型增加独立装备系统。角色当前可装备：

- **武器**：强调输出倾向与流派适配
- **衣服**：强调防御、生存或闪避
- **饰品**：强调回气、悟性或节奏辅助
- **护腕**：强调出手稳定度、臂力或局部防护
- **鞋子**：强调速度、身法与位移节奏
- **帽子**：强调命中、闪避或护头稳定性
- **戒指**：强调小幅但精确的属性补强

当前装备加成会直接进入以下战斗计算：

- 基础属性：臂力 / 身法 / 根骨 / 悟性
- 派生属性：命中 / 闪避 / 速度 / 伤害 / 防御 / 真气恢复

设计目标：

- 武学决定战斗风格
- 装备决定数值倾向
- 两者共同构成可切换流派，而不是互相替代

当前实现中，装备描述应直接说明具体属性加成，例如：

- 命中 +3%，速度 +1，伤害 +1
- 根骨 +1，闪避 +3%，防御 +1
- 回气 +1，悟性 +1

## 5. 武学系统

## 5.1 武学装备规则
角色可同时装备以下三类武学：

- **轻功**：决定身法、位移、闪避、先手与部分特殊效果
- **内功**：决定真气上限、恢复、内劲伤害、防御与被动特性
- **外功**：按兵器流派分别配置，决定主要攻击方式与战斗招式

同一时间：
- 只能装备 **1 个轻功**
- 只能装备 **1 个内功**
- 拳 / 刀 / 剑 / 棍 / 暗器各自可预设 **1 个外功**
- 当前实际生效的外功由已装备武器决定
- **未装备武器时，默认按拳法流派处理**

## 5.2 外功分类
外功可分为以下流派：

### 1. 拳法
- 偏近身连续攻击
- 适合高频出手、连击、破防
- 部分拳法附带震伤、封脉、反击效果

### 2. 剑法
- 偏均衡型流派
- 兼顾伤害、命中、连段与特殊效果
- 可设计流血、剑意、追击等机制

### 3. 刀法
- 偏高爆发与压制
- 单次伤害较高
- 可附带破甲、重创等效果

### 4. 枪棍类
- 偏控制、范围、节奏压制
- 可附带击退、迟滞、连携收益

### 5. 奇门兵器
- 如短兵、软兵、暗器系衍生流派
- 偏功能性与特殊机制
- 可作为后期稀有流派扩展

## 5.3 武学品质
建议分级：
- 凡品
- 良品
- 上乘
- 珍奇
- 绝学

## 5.4 武学获取方式
- 探索掉落
- 商店购买
- 门派兑换
- 奇遇获得
- 残页合成
- 擂台与悬赏奖励
- 隐藏高手传授

## 5.5 武学成长维度
每种武学可具备：
- 等级
- 熟练度
- 招式解锁阶段
- 特殊词条
- 领悟分支

示例：
- 初学：解锁基础招式
- 小成：解锁被动效果
- 大成：强化核心招式
- 圆满：解锁终极招式或特殊机制

## 5.6 轻功内功独立、外功随兵器切换模型
为避免武学系统在实现阶段继续扁平化，正式采用以下装备规则：

- **轻功槽**：只能装备 1 门轻功
- **内功槽**：只能装备 1 门内功
- **外功预设槽**：拳 / 刀 / 剑 / 棍 / 暗器五个流派分别保存 1 门外功
- **当前外功**：由武器决定，不再单独手动切换

三者职责划分如下：

### 轻功
- 主要提供被动属性
- 重点影响：命中、闪避、出手速度
- 不作为主要输出来源
- 可视为战斗节奏与先后手控制器

### 内功
- 以被动增益和战斗续航为主
- 重点影响：真气恢复、防御、特定外功流派加成
- 不再承担“像外功一样主动放招”的核心职责
- 内功与外功之间应存在明确的流派联动

### 外功
- 是战斗的主要主动输出来源
- 决定兵器或套路流派
- 装备后解锁对应**基础招式列表**与**绝招列表**
- 当前生效外功与武器强绑定
- 拳套对应拳法、刀对应刀法、剑对应剑法、棍对应棍法、暗器对应暗器套路
- 未装备武器时默认为空手状态，并按拳法流派结算
- 战斗中“普通攻击”不再是抽象动作，而是从当前外功的基础招式中随机施展
- 玩家主动选择的是外功下属的绝招，基础招式由系统自动调度，保证节奏简洁

该模型的设计目的：

- 降低战前操作复杂度，避免“选了武器还要单独再选当前外功”
- 保留不同兵器流派的 build 深度
- 让武器选择真正影响战斗身份，而不是只提供数值

## 5.7 外功分类与定位

### 刀法
- 强调压制、破防与正面重击
- 适合与偏伤害、偏护体的内功配合
- 更适合中低命中但高压迫的节奏

### 剑法
- 命中较稳，节奏均衡
- 适合叠加破绽、凝神、连段压制
- 适合与偏回气、偏命中的内功组合

### 拳法
- 近身压制更强
- 适合破防、连击、压低对方招架空间
- 适合与偏护体、偏硬碰硬的内功组合

### 棍法
- 擅长控距、扫线和稳定压节奏
- 适合与偏防守、偏回气的内功组合
- 可以承担更稳健的攻防转换角色

### 暗器
- 输出更偏功能性和消耗战
- 擅长流血、扰乱、迫使对手提前调息或防御
- 适合与偏命中、偏闪避、偏游斗的轻功内功组合

## 5.8 武学设计原则
参考传统武侠，尤其是金庸作品中“内功为根、轻功为骨、外功为形”的结构关系，但**不直接使用原作中的专有门派、人物、神功和招式名称**。首版原型的武学设计遵循以下原则：

- 武学名称原创，但保留经典武侠的气质
- 每门外功必须有明确兵器或套路身份
- 每门内功必须有清晰的加成方向
- 每门轻功必须能从命中、闪避、速度中至少强化两项

## 5.10 首版基础招式设计
为让普通攻击具备更明确的流派表达，所有外功都应至少配置 2 个基础招式。基础招式具有以下规则：

- 真气消耗固定为 0
- 由系统在普通攻击时随机选择
- 数值低于绝招，但能体现该外功的命中率与伤害取向
- 是外功风格的重要组成，而不是临时占位动作
- 若某个兵器流派没有装备专门外功，则回退到该流派的**基础外功**
- 未装备武器时视为空手，默认使用**基础拳脚**或已预设拳法

当前原型示例：

### 清霜江月剑
- 江风点锋：命中更稳的试探招
- 回澜扫叶：伤害更高的横削招

### 裂岳崩山拳
- 开碑直拳：均衡起手
- 沉肘锁肋：低命中高压制

### 流萤飞星手
- 弹指飞芒：高速试探暗器
- 反袖藏芒：后发重创暗器

### 秋水分波剑
- 秋锋一点：极快点刺
- 水纹横裁：轻灵横切

### 断浪沉锋刀
- 裂潮劈岸：正面沉劈，伤害更高
- 回锋抹喉：命中略低，但压制更强

### 横江伏波棍
- 横杆点波：稳健试探，利于控距
- 扫岸平沙：平扫迫位，节奏稳定

### 基础外功
- 基础拳脚：空手默认使用的基础拳法流派
- 基础刀法：未装备专门刀法时的兜底套路
- 基础剑式：未装备专门剑法时的兜底套路
- 基础棍法：未装备专门棍法时的兜底套路
- 基础暗器术：未装备专门暗器外功时的兜底套路

## 6. 战斗系统设计

## 6.1 战斗模式
- 回合制战斗
- 双方按照速度或行动条决定先后手
- 每回合选择招式、心法运转、道具、防御等行为
- 战斗节奏强调“判断、资源管理、武学克制”

## 6.2 每回合可执行行为
玩家每回合可选择：
- 普通攻击
- 使用外功绝招
- 防御 / 调息
- 使用道具
- 特殊行动（逃跑、观察、蓄力等）

补充实现规则：
- 每回合先根据**速度**决定谁先行动
- 速度主要受基础身法与已装备轻功影响
- 当前外功由武器决定；若未装备武器，则默认按拳法流派处理
- 普通攻击会从当前外功的**基础招式池**中随机选取一招
- 调息属于通用行为，但其收益主要受内功影响
- 防御属于通用行为，但其护体值主要受内功与根骨影响
- 装备会进一步修正命中、闪避、速度、伤害、防御与回气

## 6.3 战斗资源
- **气血**：归零则落败
- **真气**：释放武学消耗
- **行动顺序**：受身法、轻功、状态影响
- **状态效果**：影响战斗节奏

## 6.4 常见状态效果
- 流血：持续损失气血
- 内伤：降低内功效果或真气恢复
- 破绽：受到伤害提高
- 封脉：部分招式无法施展
- 迟滞：速度下降
- 凝神：命中或暴击提高
- 护体：减伤提升

## 6.5 克制与搭配
战斗深度主要来自以下几点：
- 轻功决定先手与闪避节奏
- 内功决定续航与特殊被动
- 外功决定输出手段与控制方式
- 装备决定数值补强方向与流派细化
- 不同敌人拥有不同抗性与弱点
- 玩家需根据敌人调整武学组合

示例：
- 高闪避敌人：适合高命中剑法或控制型枪棍
- 高护甲敌人：适合破防拳法或重击刀法
- 持久战敌人：适合高回复内功与消耗型战术
- 先手压制流：高速度轻功 + 快速基础招式 + 提速装备
- 持续消耗流：暗器外功 + 回气内功 + 命中或闪避型装备

## 6.6 首版可实现状态效果

### 1. 破绽
- 提高目标下一次所受伤害
- 适合剑法和拳法用于创造爆发窗口

### 2. 流血
- 在回合结算时持续掉血
- 适合暗器流派打消耗战

### 3. 凝神
- 提高下一次攻击的命中与伤害
- 适合作为剑法或内息蓄势的节奏点

## 7. 修炼系统

## 7.1 修炼目标
修炼是游戏核心玩法之一，目标不是单纯升级，而是：
- 提升武学境界
- 提高基础属性
- 加快真气循环
- 领悟新招式
- 形成独特流派

## 7.2 修炼方式
- **基础练功**：消耗时间提升武学熟练度
- **打坐调息**：回复状态并获取少量修为
- **闭关修炼**：集中提升某类武学或属性
- **实战领悟**：在战斗中触发突破
- **观摩残谱**：有概率领悟稀有招式

## 7.3 修炼反馈
修炼需要明确反馈：
- 数值提升
- 招式强化
- 被动解锁
- 特效变化
- 新称号或阶段描述

## 9.3 当前原型中的装备掉落逻辑
首版原型已实现基础掉落闭环，掉落结果由三部分共同决定：

- **敌人掉落表**：决定该敌人可能掉哪些装备，以及基础权重
- **场景偏向**：决定当前场景更容易产出哪种稀有度的装备
- **装备稀有度**：决定装备在掉落计算中的稀有层级

当前稀有度分为：

- 凡品
- 良品
- 珍奇

示例：

- 青石古道更偏向良品掉落
- 临河茶肆更偏向珍奇掉落
- 黑风盗的掉落表偏向暗器与轻身装备

设计意义：

- 同一名敌人在不同场景中可以掉出不同倾向的战利品
- 装备收集不再只是“从敌人身上固定拿一件”
- 后续可以自然扩展到首领专属掉落、事件掉落和区域限定掉落

图鉴作用：
- 增强收集成就感
- 提供设定说明
- 解锁少量奖励或称号

## 17. 总结

《墨影江湖》的核心不是单线叙事，而是围绕“修炼、收集、搭配、战斗、游历”建立一个可持续循环的武侠成长体验。  
玩家通过不断获取武学、调整轻功内功外功的组合，在江湖中逐步形成自己的战斗风格。  
项目首版应优先聚焦于武学系统、修炼系统与回合制战斗系统，先做出可玩的核心闭环，再逐步扩展内容体量。
